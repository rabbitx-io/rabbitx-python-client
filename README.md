## RabbitX API Python Client

### Installation

```
git clone git@gitlab.com:stripsdev/rabbit-dex-python-client.git
cd rabbit-dex-python-client
pip install .
```

### How to use

- An example using REST api client is [here](examples/rest.py)
- An example using WebSocket is [here](examples/ws.py)
